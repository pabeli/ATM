## Guía de instalación

1. Clonar el repositorio
1. Crear una base de datos en SQLServer con el nombre `ATMDW`
1. Correr sobre la base de datos el script `Scripts BD/create-tables-dw.sql`
1. Copiar el archivo `ATM/Project.params.example` en la misma ubicación y cambiar el nombre de la copia a `Project.params`
1. Editar `Project.params` para establecer las configuraciones correspondientes de cada entorno
